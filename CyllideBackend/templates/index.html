Client APIs


All the client APIs apart from otpsender and otpverifier take in and send out base64 encrypted responses. They all accept and send encrypted json objects. In order to be able to use them, contact me to recieve the development key, that has to kept in the app, in order to be able to encrypt and decrypt server responses.


Authorization Procedure: The verify OTP method will return you a json web token that has to be passed in the header of all the other API calls in order to authenticate your request. It is recommended that you store it in shared preferences
